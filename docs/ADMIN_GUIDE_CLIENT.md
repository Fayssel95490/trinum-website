# Guide d'utilisation de l'espace administration

Bienvenue ! Ce guide vous explique comment modifier vous-même le contenu de votre site trinum-ingenierie.fr.

## Se connecter

1. Rendez-vous sur **https://[votre-domaine]/admin**
2. Saisissez votre identifiant et votre mot de passe (fournis par votre développeur)
3. Cliquez sur **Se connecter**

## Les 4 onglets

Une fois connecté, vous avez 4 onglets en haut :

### 1. Accueil

Toutes les sections de la page d'accueil :

- **Hero** : le grand bloc en haut (titre, sous-titre, image)
- **Pourquoi Trinum** : les 4 arguments mis en avant
- **Image pleine largeur** : la grande photo entre les sections
- **Étapes du projet** : les 4 étapes (Cadrage, Avant-projet…)
- **Références & Réalisations** : le titre de la section projets
- **Appel à l'action** : le bloc "Évaluez votre projet" en bas

### 2. Projets

La liste de vos fiches projets. Vous pouvez :

- **Ajouter** un nouveau projet (bouton en haut à droite)
- **Éditer** un projet existant (bouton crayon)
- **Supprimer** un projet (bouton poubelle rouge)
- **Réordonner** la liste (flèches haut/bas)

Dans l'éditeur d'un projet, vous pouvez modifier tous les champs : titre, sous-titre, tags, images, contexte, défis, citation, sections techniques avec mise en forme (gras, italique, listes…), synthèse, conclusion.

### 3. Contact & Calendly

- **URL Calendly** : le lien vers votre page de prise de rendez-vous
- Textes de la page contact
- Types de projets proposés dans le formulaire

### 4. Infos société

- Nom et tagline
- Email, téléphone, LinkedIn
- Menu de navigation (liens en haut du site)
- Logos partenaires ("Trusted by")

## Comment modifier

### Modifier un texte

Cliquez dans un champ et tapez directement. La barre en bas affiche **"Modifications non enregistrées"**.

### Changer une image

1. Cliquez sur **Changer l'image** à côté de l'image actuelle
2. Choisissez un fichier depuis votre ordinateur (JPEG, PNG, WebP ou GIF, max 5 Mo)
3. L'image s'upload automatiquement et s'affiche

### Ajouter / supprimer un élément (ex : un partenaire, un point fort)

- **Ajouter** : bouton **+ Ajouter…** en bas de la liste
- **Supprimer** : bouton poubelle rouge à côté de l'élément
- **Réordonner** : flèches haut/bas

### Enregistrer

En bas de l'écran, une barre fixe affiche le statut :

- **"À jour"** : rien à sauvegarder
- **"Modifications non enregistrées"** : cliquez sur **Enregistrer et publier**

Après enregistrement, un message de confirmation apparaît. **Vos modifications sont visibles sur le site dans environ 1 minute.**

## Bonnes pratiques

- **Prévisualisez avant de publier** : ouvrez le site dans un autre onglet, modifiez dans l'admin, enregistrez, puis rafraîchissez l'autre onglet après 1 min.
- **Images** : évitez les photos trop grandes. Le format idéal est autour de 1920×1080 px, JPEG ou WebP.
- **Texte** : restez concis, notamment pour le hero et les sous-titres.
- **Enregistrez régulièrement** : chaque onglet a son propre bouton d'enregistrement, pensez à sauvegarder avant de changer d'onglet.

## Déconnexion

Cliquez sur **Déconnexion** en haut à droite.

## En cas de problème

- **Le site ne reflète pas mes changements** : attendez 1 minute puis rafraîchissez la page (Ctrl+F5 / Cmd+Shift+R).
- **Message d'erreur à l'enregistrement** : notez le message et contactez votre développeur.
- **Identifiants perdus** : contactez votre développeur pour qu'il regénère un mot de passe.

Bonne édition !
